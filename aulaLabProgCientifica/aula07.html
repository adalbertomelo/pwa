Aula: Linguagens de Programação Estruturadas

Disciplina: Laboratório de Programação Científica | Curso: Ciência da Computação

Objetivo da Aula

  • Compreender os fundamentos das linguagens de programação estruturadas.
  • Escrever programas simples usando estruturas básicas (sequência, decisão, repetição).
  • Resolver problemas aplicados com exemplos práticos.

O que é Programação Estruturada?

Definição: Paradigma baseado em dividir o programa em blocos ou estruturas.

  • Uso de sequências, condicionais e loops.
  • Foco em código legível, modular e fácil de depurar.

Características principais:

  1. Controle linear de execução.
  2. Uso de sub-rotinas ou funções.
  3. Modularidade.

Estruturas Básicas

  • Sequência: Execução de comandos em ordem.
  • Decisão (Condicional): if, else, elif.
  • Repetição (Laços): for, while.

Exemplo 1 - Sequência

# Calcular a soma de dois números
num1 = 5
num2 = 10
soma = num1 + num2
print("A soma é:", soma)

Atividade: Modifique o código para calcular a média de três números.

Exemplo 2 - Decisão

# Verificar se um número é par ou ímpar
numero = int(input("Digite um número: "))
if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

Atividade: Adicione uma verificação para números negativos.

Exemplo 3 - Repetição com for

# Imprimir os números de 1 a 10
for i in range(1, 11):
    print(i)

Atividade: Altere o código para imprimir os números pares entre 1 e 20.

Exemplo 4 - Repetição com while

# Somar números até que o usuário digite 0
soma = 0
while True:
    numero = int(input("Digite um número (0 para sair): "))
    if numero == 0:
        break
    soma += numero
print("A soma total é:", soma)

Atividade: Altere o código para exibir a média dos números digitados.

Modularização com Funções

def somar(a, b):
    return a + b

num1 = 4
num2 = 6
print("A soma é:", somar(num1, num2))

Atividade: Crie uma função para calcular o fatorial de um número.

Projeto Prático Integrado

Descrição: Desenvolver um programa que simula um caixa eletrônico.

  • Exibir um menu com as opções: Saldo, Saque, Depósito e Sair.
  • Usuário deve conseguir realizar operações até escolher "Sair".

Dica: Use loops, condicionais e funções para modularizar o código.

Resumo e Conclusão

  • Estruturas básicas: Sequência, Decisão, Laços.
  • Modularização com funções.
  • Aplicação prática com projeto integrado.
