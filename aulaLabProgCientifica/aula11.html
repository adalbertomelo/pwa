Aula: Gráficos em Python

Introdução

Gráficos são ferramentas essenciais para a análise e visualização de dados. Python oferece várias bibliotecas poderosas para criar gráficos. Nesta aula, exploraremos as principais bibliotecas: Matplotlib, Seaborn, Plotly e Bokeh.

1. Matplotlib

O Matplotlib é uma biblioteca básica e flexível para visualização de dados em Python.

Tipos de gráficos suportados

  • Gráficos de linha
  • Gráficos de barra
  • Gráficos de dispersão
  • Gráficos de pizza
  • Histogramas

Exemplo: Gráfico de linha

import matplotlib.pyplot as plt
x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 40]
plt.plot(x, y)
plt.title("Gráfico de Linha")
plt.xlabel("Eixo X")
plt.ylabel("Eixo Y")
plt.show()

Exemplo: Gráfico de dispersão

import matplotlib.pyplot as plt
x = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11]
y = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78]
plt.scatter(x, y)
plt.title("Gráfico de Dispersão")
plt.show()

2. Seaborn

O Seaborn fornece uma interface de alto nível para criar gráficos estatísticos atraentes.

Tipos de gráficos suportados

  • Gráficos de densidade
  • Gráficos de violino
  • Gráficos de caixa
  • Gráficos de calor (heatmaps)

Exemplo: Gráfico de calor

import seaborn as sns
import numpy as np
data = np.random.rand(10, 10)
sns.heatmap(data, annot=True)
plt.title("Gráfico de Calor")
plt.show()

3. Plotly

O Plotly é uma biblioteca interativa para criar gráficos sofisticados.

Tipos de gráficos suportados

  • Gráficos 3D
  • Gráficos de superfície
  • Mapas

Exemplo: Gráfico 3D

import plotly.express as px
df = px.data.iris()
fig = px.scatter_3d(df, x='sepal_length', y='sepal_width', z='petal_length',
color='species')
fig.show()

4. Bokeh

O Bokeh é ideal para criar gráficos interativos em navegadores.

Tipos de gráficos suportados

  • Gráficos de linhas
  • Gráficos de dispersão interativos
  • Gráficos em tempo real

Exemplo: Gráfico de linha

from bokeh.plotting import figure, show
p = figure(title="Exemplo Bokeh")
p.line([1, 2, 3, 4], [10, 20, 25, 30])
show(p)
