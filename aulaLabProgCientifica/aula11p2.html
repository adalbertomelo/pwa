Aula: Gráficos com Matplotlib

Introdução

O Matplotlib é uma biblioteca essencial para criar visualizações de dados em Python. Nesta aula, exploraremos os principais tipos de gráficos suportados pela biblioteca com exemplos práticos.

1. Gráficos de Linha

Os gráficos de linha são usados para mostrar tendências ao longo do tempo ou relações contínuas entre dados.

Exemplo:

import matplotlib.pyplot as plt
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]
plt.plot(x, y, marker='o')
plt.title("Gráfico de Linha")
plt.xlabel("Eixo X")
plt.ylabel("Eixo Y")
plt.grid(True)
plt.show()

2. Gráficos de Barra

Gráficos de barra são usados para comparar categorias ou grupos de dados.

Exemplo:

import matplotlib.pyplot as plt
categorias = ['A', 'B', 'C', 'D']
valores = [5, 7, 3, 8]
plt.bar(categorias, valores, color='skyblue')
plt.title("Gráfico de Barra")
plt.xlabel("Categorias")
plt.ylabel("Valores")
plt.show()

3. Gráficos de Dispersão

Gráficos de dispersão são usados para observar relações ou padrões entre duas variáveis.

Exemplo:

import matplotlib.pyplot as plt
x = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11]
y = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78]
plt.scatter(x, y, color='red')
plt.title("Gráfico de Dispersão")
plt.xlabel("X")
plt.ylabel("Y")
plt.show()

4. Gráficos de Pizza

Gráficos de pizza são usados para mostrar proporções ou partes de um todo.

Exemplo:

import matplotlib.pyplot as plt
labels = ['A', 'B', 'C', 'D']
sizes = [30, 45, 15, 10]
plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=140)
plt.title("Gráfico de Pizza")
plt.axis('equal') # Para manter o círculo perfeito
plt.show()

5. Histogramas

Histogramas são usados para visualizar a distribuição de dados.

Exemplo:

import matplotlib.pyplot as plt
import numpy as np
data = np.random.randn(1000)
plt.hist(data, bins=20, color='purple', alpha=0.7)
plt.title("Histograma")
plt.xlabel("Valores")
plt.ylabel("Frequência")
plt.show()
