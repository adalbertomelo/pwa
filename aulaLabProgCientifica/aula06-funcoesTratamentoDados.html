Aula: Funções de Tratamento de Dados em Python

Objetivo da Aula

  • Introduzir as principais funções e bibliotecas de Python usadas para tratamento de dados.
  • Demonstrar exemplos práticos de manipulação de dados.

1. Introdução

O que é tratamento de dados?

Tratamento de dados envolve limpar, formatar, transformar e manipular dados para análise.

Bibliotecas principais em Python:

  • pandas: Manipulação e análise de dados tabulares.
  • numpy: Manipulação de arrays e operações matemáticas.
  • re: Tratamento de strings com expressões regulares.
  • datetime: Manipulação de datas e horários.

2. Importação das Bibliotecas

import pandas as pd
import numpy as np
import re
from datetime import datetime

3. Manipulação de Strings

Remover espaços em branco:

string = " exemplo de string "
print(string.strip()) # Remove espaços no início e no fim

Alterar capitalização:

print(string.lower()) # Tudo em minúsculas
print(string.upper()) # Tudo em maiúsculas

Substituir partes de uma string:

nova_string = string.replace("exemplo", "demonstração")
print(nova_string)

Uso de expressões regulares (re):

texto = "Meu número é 123-456-7890"
padrao = r"\d{3}-\d{3}-\d{4}"
if re.search(padrao, texto):
    print("Número de telefone encontrado!")

4. Tratamento de Dados Numéricos com numpy

Criar arrays:

array = np.array([1, 2, 3, 4, 5])
print(array)

Operações matemáticas:

print(np.mean(array)) # Média
print(np.median(array)) # Mediana
print(np.std(array)) # Desvio padrão

5. Manipulação de Datas com datetime

Criar objetos de data:

data = datetime(2024, 10, 31)
print(data.strftime("%d/%m/%Y")) # Formatar data como string

Conversão de strings para datas:

data_string = "31-10-2024"
data_objeto = datetime.strptime(data_string, "%d-%m-%Y")
print(data_objeto)

Diferença entre datas:

data1 = datetime(2024, 10, 31)
data2 = datetime(2024, 10, 25)
diferenca = data1 - data2
print(diferenca.days) # Dias de diferença

6. Manipulação de Dados com pandas

Criar um DataFrame:

dados = {"Nome": ["Ana", "Bruno", "Carlos"],
    "Idade": [25, 30, 35],
    "Cidade": ["Belém", "Ananindeua", "Castanhal"]}
df = pd.DataFrame(dados)
print(df)

Limpar valores ausentes:

df["Idade"] = df["Idade"].fillna(df["Idade"].mean()) # Substituir NaN pela média

Filtrar dados:

df_filtrado = df[df["Idade"] > 28]
print(df_filtrado)

Alterar e formatar colunas:

df["Idade"] = df["Idade"].apply(lambda x: x + 1) # Incrementar todas as idades em 1
print(df)
