Programas Envolvendo Processos de Repetição e Seleção

Curso de Bacharelado em Ciência da Computação

Objetivos da Aula

  • Compreender os conceitos de estruturas de repetição (for, while, do-while).
  • Entender as estruturas de seleção (if-else, switch).
  • Aplicar os conceitos em problemas reais e construir programas eficientes.

Conteúdo Programático

Parte 1: Introdução

Estruturas de repetição e seleção são fundamentais para a construção de programas dinâmicos e interativos. Elas permitem executar blocos de código múltiplas vezes ou tomar decisões baseadas em condições.

Parte 2: Estruturas de Seleção

  • If-Else: Condicional básica para bifurcação.
  • Switch: Ideal para múltiplas condições.

Parte 3: Estruturas de Repetição

  • For: Estrutura de repetição definida.
  • While: Estrutura de repetição condicional.
  • Do-While: Estrutura condicional que garante ao menos uma execução.

Exemplos

Exemplo de Estrutura de Seleção

Programa para verificar se um número é positivo, negativo ou zero:


int num;
printf("Digite um número: ");
scanf("%d", &num);

if (num > 0) {
    printf("O número é positivo\n");
} else if (num < 0) {
    printf("O número é negativo\n");
} else {
    printf("O número é zero\n");
}

Exemplo de Estrutura de Repetição

Programa para calcular a soma dos números de 1 a 10 usando for:


int soma = 0;
for (int i = 1; i <= 10; i++) {
    soma += i;
}
printf("A soma é: %d\n", soma);

Exercícios Práticos

Nível Simples

Exercício 1

Escreva um programa que exiba todos os números de 1 a 10 usando um loop for.

Exercício 2

Crie um programa que leia um número e exiba se ele é par ou ímpar usando if-else.

Exercício 3

Implemente um programa que exiba "Hello World" 5 vezes usando while.

Nível Intermediário

Exercício 4

Escreva um programa que leia um número e exiba a soma de todos os números de 1 até o número fornecido.

Exercício 5

Crie um programa que leia 10 números e exiba o maior e o menor entre eles.

Exercício 6

Construa um programa que exiba a tabuada de um número fornecido pelo usuário.

Nível Avançado

Exercício 7

Desenvolva um programa que simule uma calculadora simples. O usuário deve informar dois números e a operação desejada (adição, subtração, multiplicação, divisão). Utilize switch para as operações.

Exercício 8

Implemente um programa que leia 5 números e exiba apenas os números primos.

Exercício 9

Escreva um programa que leia uma palavra e a exiba de trás para frente.

Exercício 10

Desenvolva um programa que leia uma matriz 3x3 de números inteiros e calcule a soma dos elementos da diagonal principal.
