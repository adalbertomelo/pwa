Aula: Gráficos com Seaborn

Introdução

O Seaborn é uma biblioteca poderosa para criar gráficos estatísticos e visualizações atraentes em Python. Nesta aula, exploraremos gráficos de densidade, violino, caixa e calor (heatmaps) com exemplos práticos.

1. Gráficos de Densidade

Os gráficos de densidade são usados para representar a distribuição de dados de maneira suave.

Exemplo:

import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np
data = np.random.randn(1000)
sns.kdeplot(data, shade=True, color="blue")
plt.title("Gráfico de Densidade")
plt.show()

2. Gráficos de Violino

Os gráficos de violino mostram a distribuição dos dados juntamente com sua densidade.

Exemplo:

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
data = sns.load_dataset("tips")
sns.violinplot(x="day", y="total_bill", data=data, palette="muted")
plt.title("Gráfico de Violino")
plt.show()

3. Gráficos de Caixa

Os gráficos de caixa mostram os quartis, a mediana e os outliers de um conjunto de dados.

Exemplo:

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
data = sns.load_dataset("tips")
sns.boxplot(x="day", y="total_bill", data=data, palette="coolwarm")
plt.title("Gráfico de Caixa")
plt.show()

4. Gráficos de Calor (Heatmaps)

Os heatmaps são usados para visualizar a correlação ou concentração de valores em uma matriz de dados.

Exemplo:

import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np
data = np.random.rand(10, 10)
sns.heatmap(data, annot=True, cmap="YlGnBu")
plt.title("Gráfico de Calor")
plt.show()
