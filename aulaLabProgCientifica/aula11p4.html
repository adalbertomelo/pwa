Aula: Gráficos com Plotly

Introdução

O Plotly é uma biblioteca interativa para criação de gráficos avançados em Python. Nesta aula, exploraremos gráficos 3D, gráficos de superfície e mapas com exemplos práticos e interativos.

1. Gráficos 3D

Gráficos 3D permitem visualizar dados em três dimensões, oferecendo uma perspectiva mais rica sobre os dados.

Exemplo:

import plotly.express as px
import pandas as pd
df = px.data.iris()
fig = px.scatter_3d(df, x='sepal_length', y='sepal_width', z='petal_length',
color='species', size='petal_width',
title='Gráfico 3D - Iris Dataset')
fig.show()

2. Gráficos de Superfície

Gráficos de superfície são úteis para representar funções matemáticas ou modelos de dados em 3D.

Exemplo:

import plotly.graph_objects as go
import numpy as np
x = np.linspace(-2, 2, 100)
y = np.linspace(-2, 2, 100)
x, y = np.meshgrid(x, y)
z = np.sin(np.sqrt(x**2 + y**2))
fig = go.Figure(go.Surface(z=z, x=x, y=y, colorscale='Viridis'))
fig.update_layout(title='Gráfico de Superfície', scene=dict(zaxis=dict(range=[-1, 1])))
fig.show()

3. Mapas

Mapas em Plotly são úteis para representar dados geoespaciais de maneira interativa.

Exemplo:

import plotly.express as px
df = px.data.gapminder()
fig = px.choropleth(df, locations="iso_alpha", color="lifeExp",
hover_name="country", animation_frame="year",
title="Expectativa de Vida Global")
fig.show()
